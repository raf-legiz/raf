# Обновления
## Проверяем обновление, желательно проверять каждую неделю. 

Обновить список пакетов

Посмотреть доступные security-обновления
```shell
apt list --upgradable 2>/dev/null | grep -i security
```
Если есть критические обновления, установить их
```shell
sudo apt upgrade -y
```


## Ежемесяное или квартальное обслуживание сервера 

Полное обновление
```shell
sudo apt update && sudo apt full-upgrade -y
```

Удаление ненужных пакетов

```shell
sudo apt autoremove -y
```

Очистка кеша
```shell
sudo apt clean
```

Проверка необходимости перезагрузки
```shell
if [ -f /var/run/reboot-required ]; then
    echo "Требуется перезагрузка"
    sudo reboot
fi
```
## обновление версии LTS → LTS

1. Полностью обновить текущую систему
```shell
sudo apt update && sudo apt full-upgrade -y
```

2. Убедиться, что установлен update-manager-core
```shell
sudo apt install update-manager-core
```

3. Проверить настройки 
```shell
/etc/update-manager/release-upgrades (Prompt=lts)
```

4. Проверить доступность новой версии
```shell
sudo do-release-upgrade -c
```

5. Запустить обновление
```shell
sudo do-release-upgrade
```
